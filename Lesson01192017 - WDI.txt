Working Directory: Parent of ALL
 - parent: individual folder hosting a child 
	- child: file inside a folder

Git - version control software
commit - to save point
- to make a commit is to snapshot
- give it a message
- commit Early, Commit ofren

Branching
- is useful when you do not want to break anything from original

ABOUT GIT-ING
"git init" - initializes our folder

"git add" - to add files to our staging area

"git commit -m" - initial commit

"git status" - tracks status of what I've added and what I haven't

"git push origin master" - to put changes from local to online

"git pull origin master" - to bring changes in from online to local

"remove .git" - to delete a repository locally *do this with Caution!*

version - a way to track all changes and view these changes.

repository - is a place where you save all commits

CHANGE A REMOTE URL
